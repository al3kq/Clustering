K-Means:¶

K-means minimizes the distances within a cluster and maximizes the distance between clusters.

Random points are chosen and clusters of nearest points are made. A centriod is found from these clusters and this process is repeated until the clusters don't change.

A flaw in this method is the number of clusters must be predetermined.

import matplotlib.pyplot as plt
from sklearn import datasets
import numpy as np
from sklearn.cluster import KMeans

# Importing a loading Iris dataset
X, y = datasets.load_iris(return_X_y = True)

# Three clusters are made for each flower category
model = KMeans(n_clusters=3)
model.fit(X)
y_labels = model.predict(X)

Evaluating a Cluster 2¶

For unlabeled data, look at how spread out the clusters are. Use the distance from each sample to centroid of its cluster. This is the inertia metric. Adding more clusters will lower the inertia, but you can have too many clusters. Choose a cluster number with high marginal change.

In [119]:
ks = range(1,6)
inertias = []
for k in ks:
  model = KMeans(n_clusters= k)
  model.fit(X)
  inertias.append(model.inertia_)

plt.plot(ks, inertias, '-o')
plt.xlabel("Number of Clusters")
plt.ylabel("Inertia")
plt.show

Transforming Data for Clusters¶

Clustering will be much more accurate and interpretable with scaled features. Use the StandardScaler to get equal variance.

Visualizing Clusters¶

Agglomerative Hierarchy clustering: Every point has it's own cluster, at each step 2 closest clusters are merged. Visualize with dendogram tree.

In [122]:
from scipy.cluster.hierarchy import linkage, dendrogram
mergings = linkage(newdata, method = 'complete')
dendrogram(mergings, labels = target_wine)
plt.show()

Hierarchical Clustering¶

Extract clusters from intermediate stage. Find intermediate clusters by defining a dendrogram height. This height is the max distance bewtween points. Distance is defined by linkage method ("complete").

DBScan¶

Specify how close things have to be and minimum points in a cluster. Don't need to define the number of clusters

In [125]:
from sklearn.cluster import DBSCAN
iris = datasets.load_iris()
X = iris.data[:, :2]
y = iris.target

db = DBSCAN(eps = 0.3, min_samples = 10)

labels = db.labels_

x_labels = ['setosa','versicolor','virginica']
fig, ax = plt.subplots()
